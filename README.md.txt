# Projet 1 — Analyse des Ventes Retail 

![Dashboard](rapport_visuel_retail.png)

## Contexte
Une entreprise e-commerce britannique souhaite mieux comprendre
ses ventes sur la période 2010-2011. Cette analyse EDA complète
explore les tendances de revenus, les marchés internationaux,
les produits phares et les patterns d'achat des clients.

## Dataset
- **Source :** [UCI Online Retail Dataset](https://archive.ics.uci.edu/dataset/352/online+retail)
- **Dimensions :** 541 909 transactions → 530 104 après nettoyage
- **Période :** Décembre 2010 – Décembre 2011
- **Marché :** E-commerce UK + 37 pays internationaux

## Insights clés

| # | Finding | Impact |
|---|---|---|
|   | Le Royaume-Uni représente **87% du CA total** (£9M sur £10.6M) | Marché domestique dominant, international sous-exploité |
|   | **Novembre 2011** est le mois record avec £1.45M de CA | Saisonnalité Noël forte — stock et marketing à anticiper |
|   | Les achats se concentrent **mardi/jeudi entre 10h et 15h** | Clientèle majoritairement **B2B** (professionnels) |
|   | Pays-Bas et EIRE dominent l'international (**21% chacun**) | Marchés prioritaires pour une stratégie d'expansion |
|   | Top produit : **REGENCY CAKESTAND 3 TIER** (£174K) | Catalogue cohérent cadeaux & décoration maison |

## Recommandations business
1. **Développer l'international** — 13% du CA seulement, fort potentiel de croissance
2. **Anticiper la saisonnalité** — constituer les stocks dès septembre pour le pic de novembre
3. **Programme fidélité B2B** — les 10 meilleurs clients génèrent une part disproportionnée du CA
